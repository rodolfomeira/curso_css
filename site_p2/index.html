Loz Polemicos é o grupo fundador do site “Loz Filmes” Fundamos o site com o propósito de trazer praticidade e informação no momento de pesquisar os incríveis filmes que irão lançar nos cinemas, tendo seus trailer já em nossa página para que voce não precise estar se deslocando de uma página para outra, trazendo, tambem, a data de lançamento de cada filme. Confira em cada aba a cima os filmes separados por gêneros.

Sobre nós:

Formamos o grupo a partir da ideia de fazer projetos diferenciados, causando polemicas a partir da característica individual de cada integrante: Khevyn como “O Eremita”, Leonardo como “O Prodígio”, Gabriel como “O Artista”, Samira como “A Genial” e Jhonattan como “O Capitão”. Nós somos os: Loz Polemicos.
